Notes I found About Tags

<h1> </h1>
This is the main heading tag that is used in the header.
<p></p>
This tag should be used when adding paragraphs to the webpage.
<cite></cite>
This tag should be used when citing from another source, often used with quotations.
<em></em>
This tag should be used when adding emphasis to text. Usually for more important text, like this: Important Text.
<sub></sub>
This tag should be used when adding subscript text like this: H2O.
